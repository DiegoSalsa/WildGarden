# Floreria WildGarden - Tienda Online

Sitio web e-commerce para **WildGarden** - Floristería con diseño natural, silvestre y elegante.

## Frontend
- **HTML5** - Estructura semántica
- **CSS3** - Diseño responsivo con Flexbox/Grid
- **JavaScript** - Interactividad (menú, búsqueda, carrito)
- **Tipografía:** EB Garamond (elegancia clásica en toda la web)

## Identidad Visual
- **Estilo:** Natural, Silvestre y Elegante
- **Paleta de Colores:**
  - Verde Muy Oscuro: `#253020` (Texto principal y botones)
  - Verde Medio: `#4C6443` (Acentos y bordes)
  - Crema: `#EDDECB` (Fondos y secciones destacadas)
  - Blanco: `#FFFFFF` (Fondo limpio)
